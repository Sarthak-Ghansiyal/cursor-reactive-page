# Cursor Reactive Page

An interactive webpage that responds to your cursor in real time — built with pure HTML, CSS, and vanilla JavaScript. No libraries. No frameworks.

## Live Demo
🔗 [cursor-reactive-page](https://sarthak-ghansiyal.github.io/cursor-reactive-page)

## Features
- Custom animated cursor with a lagging ring effect
- Proximity-based tile grid that lights up around your mouse
- Particle trail that follows cursor movement
- Magnetic button that pulls toward your cursor
- Ambient glow that smoothly tracks mouse position

## What I Learned
- `mousemove` event handling and real-time DOM updates
- `requestAnimationFrame` for smooth 60fps animations
- CSS `mix-blend-mode` and custom cursor techniques
- Magnetic UI / physics-based interactions
- Building a particle system from scratch
